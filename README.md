## Definitions

- **`user`** -

  - **Type** - object
  - **Required Properties:**
    - userId
    - taskId
    - username
    - password
  - **Properties:**
    - **userId** -
      - **Type** - integer
      - **Format** - int64
    - **username** -
      - **Type** - string
    - **firstName** -
      - **Type** - string
    - **lastName** -
      - **Type** - string
    - **email** -
      - **Type** - string
    - **password** -
      - **Type** - string

- **`task`** -
  - **Type** - object
  - **Required Properties** -
    - taskId
    - userId
    - taskName
  - **Properties** -
    - **TaskId** -
      - **Type** - integer
      - **Format** - int64
    - **userId** -
      - **Type** - integer
      - **Format** - int64
    - **taskName** -
      - **Type** - string
    - **taskDueDate** -
      - **Type** - string
      - **Format** - date-time
    - **taskStatus** -
      - **Type** - string
      - **Format** -
        - enum (array(3)) -
          - created
          - in progress
          - done
        - **default** - created
    - **complete** -
      - **Type** - boolean
      - **default** - false
